# github-host-modify-tool
自动修改github网站上图片域名的脚本

Quick Start:
<!-- pm2可以将程序常驻在后台 -->
```$ npm install pm2 -g```

启动脚本

```$ pm2 start xxx/tools/github-host-modify-tool --name <app_name>```

停止脚本

```$ pm2 stop <app_name>```

删除脚本

```$ pm2 delete <app_name>```

更多指令参考[pm2](https://pm2.keymetrics.io/docs/usage/pm2-doc-single-page/)
